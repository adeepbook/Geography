## 板块/汇聚/洋陆俯冲
- 一句话: 大洋板块俯冲到大陆/岛弧之下,生成海沟 + 平行的火山山脉。
- 原理讲解: 致密的大洋板块下插入较轻的大陆板块之下,下插处海底拗成海沟;板片下沉到一定深度释放水分,促使上覆地幔熔融,岩浆上升在大陆边缘排出一列火山(火山弧)。深源地震沿下插板片成带分布。
- 剖面画法: 左大洋板块向右下斜插,海沟处下凹;上覆板块边缘画一列火山;板片上画震源点带。
- 动画脚本: ①大洋板块逼近→②下插成沟→③板片脱水、上覆地幔熔融→④火山弧喷发。
- 参数槽: [俯冲板块, 上覆板块, 火山弧代表]
- 易错警示: 俯冲带火山岩浆来自上覆地幔的熔融,不是"下插板块本身熔化喷出"。
- 状态: 起草

## 板块/汇聚/陆陆碰撞
- 一句话: 两块大陆相撞,洋壳耗尽后地壳叠置加厚,隆起巨型山脉。
- 原理讲解: 两陆间的大洋先经俯冲消耗殆尽,随后两块都太轻、谁也插不下去,于是地壳相互叠覆、挤压加厚(可达正常两倍),夹在中间的海相沉积被逆冲、褶皱并整体抬升——昔日海底升上山巅。喜马拉雅—青藏是范本。
- 剖面画法: 两大陆相向挤压,中间地壳叠瓦状增厚抬升;标海相地层被抬高的位置。
- 动画脚本: ①洋盆俯冲缩小→②两陆相接→③地壳叠覆加厚→④逆冲抬升、海相岩登顶。
- 参数槽: [碰撞双方, 起始时代, 地壳增厚幅度]
- 易错警示: 山顶海相岩=地层被抬上去,不是海水曾漫到山顶。
- 状态: 起草

## 板块/离散/大陆裂谷
- 一句话: 大陆被拉开,正断层成排陷落成谷,裂谷是未来海洋的胚胎。
- 原理讲解: 地幔上涌使大陆穹起、变薄、受拉张,成排正断层让中间断块陷落成长条地堑;沿裂缝减压熔融生成火山,洼地积水成裂谷湖。若持续拉张,海水终将灌入,裂谷转为新生洋盆(红海即此阶段)。
- 剖面画法: 中央地堑下陷,两侧断阶式抬升的裂谷肩;下方地幔上涌;谷中画湖与火山。
- 动画脚本: ①地幔上涌穹起→②拉张、正断层陷落→③裂谷湖与火山→④(分支)海侵成新洋。
- 参数槽: [裂解板块对, 阶段(早/晚), 代表火山与湖]
- 易错警示: 裂谷是"拉开"的产物(张性),与挤压造山相反。
- 状态: 起草

## 岩石/火成/安山岩
- 一句话: 中性岩浆在俯冲带火山喷出,是成层火山的主要岩石。
- 原理讲解: 成分介于玄武岩与流纹岩之间,黏度较高、含气较多,喷发常较剧烈;典型产在大陆边缘与岛弧的俯冲带火山(环太平洋"安山岩线")。颜色多灰,常见斑状结构。
- 剖面画法: 火山锥内灰色熔岩与碎屑互层;斑晶可示意。
- 动画脚本: ①弧岩浆上升→②黏稠岩浆爆发式喷出→③熔岩与碎屑交替堆积。
- 参数槽: [产出火山, 喷发风格, 时代]
- 易错警示: 富士山岩浆偏玄武质,属特例;"安山=所有俯冲火山"需按地核对。
- 状态: 起草

## 板块/火山/成层火山
- 一句话: 中酸性岩浆熔岩与碎屑互层堆出陡峭锥形,喷发剧烈。
- 原理讲解: 黏稠岩浆既溢流又爆发,熔岩流与火山碎屑(火山灰、火山弹)交替成层,堆出坡度大、对称度高的标准火山锥;喷发猛烈,常伴火山碎屑流。与盾状火山的缓坡形成鲜明对比。
- 剖面画法: 陡而对称的锥;内部熔岩层与碎屑层交替;顶部火口。
- 动画脚本: ①爆发喷碎屑→②溢流铺熔岩→③层层交替增高→④对称锥成形(对比插卡:盾状缓坡)。
- 参数槽: [代表火山, 岩性, 喷发史]
- 易错警示: "成层/盾状"指形态与喷发风格之别,根源在岩浆黏度,不是规模大小。
- 状态: 起草

## 构造/正断层
- 一句话: 拉张环境,上盘沿断面下滑;裂谷与断陷盆地的基本单元。
- 原理讲解: 水平拉张使岩层沿倾斜断面错动,上盘相对下降。成组出现即造出地堑(陷落谷)与地垒(翘起块)。是离散边界与伸展构造区的标志。
- 剖面画法: 倾斜断面,上盘下滑、地层错开;箭头示相对位移。
- 动画脚本: ①拉张→②断面破裂→③上盘下滑、地层错断。
- 参数槽: [断距, 倾角, 构造背景]
- 易错警示: 正断层=拉张(上盘下降);逆断层=挤压(上盘上升),勿混。
- 状态: 起草

## 构造/逆断层(含叠瓦逆冲)
- 一句话: 挤压环境,上盘被推覆上升;造山带的标配。
- 原理讲解: 水平挤压使上盘沿断面被推上去、相对升高;低角度大位移者为逆冲断层,多条叠置成叠瓦状(如喜马拉雅 MFT/MBT/MCT),把深部老岩推覆到浅部新岩之上,造成地壳叠覆加厚。
- 剖面画法: 倾斜断面,上盘上冲叠覆;多条平行叠瓦。
- 动画脚本: ①挤压→②上盘上冲→③叠瓦堆叠、地壳加厚抬升。
- 参数槽: [断距, 倾角, 叠瓦条数]
- 易错警示: 逆冲会让"老岩盖在新岩之上",是构造倒置,不是沉积顺序。
- 状态: 起草

## 构造/地堑与地垒
- 一句话: 正断层成组出现:中间陷落为地堑,两侧翘起为地垒。
- 原理讲解: 区域拉张下,平行正断层把地壳切成断块,相对下降的中央块体为地堑(成谷、常积水),相对抬升的块体为地垒(成岭)。东非裂谷、汾渭地堑是典型。
- 剖面画法: 中央块下陷、两侧块抬起的阶梯断块。
- 动画脚本: ①拉张→②成组正断层→③中陷成谷、侧翘成岭。
- 参数槽: [宽度, 断块数, 充水情况]
- 易错警示: 地堑由拉张造成,不是被两侧"挤"下去的。
- 状态: 起草

## 构造/背斜
- 一句话: 岩层受挤上拱,核老翼新;可成山,顶部张裂后也可被蚀成谷。
- 原理讲解: 水平挤压使岩层向上弯成拱,核部出露较老地层。背斜顶部受张力易生裂隙、被优先侵蚀,故"背斜成谷、向斜成山"的地形倒置并不少见。
- 剖面画法: 向上凸的弧形层;核部标老地层;可示顶部张裂。
- 动画脚本: ①水平层→②挤压上拱→③顶部张裂、差异侵蚀。
- 参数槽: [核部时代, 规模, 现今地形(山/谷)]
- 易错警示: 背斜≠一定成山;地形受后期侵蚀控制,可倒置。
- 状态: 起草

## 构造/向斜
- 一句话: 岩层受挤下凹,核新翼老;槽部坚实,常反而成山。
- 原理讲解: 挤压使岩层向下弯成槽,核部为较新地层。槽部岩石受压紧实、抗蚀,常在长期侵蚀后反而保留成山(向斜山),与背斜谷成对出现。
- 剖面画法: 向下凹的弧形层;核部标新地层。
- 动画脚本: ①水平层→②挤压下凹→③差异侵蚀、向斜成山。
- 参数槽: [核部时代, 规模, 现今地形]
- 易错警示: 与背斜配套理解;别只记"向下凹"而忘了地形倒置。
- 状态: 起草

## 外力/流水/先成河
- 一句话: 河比山老:山慢慢隆起,河保持原路下切,硬生生锯穿山脉。
- 原理讲解: 当河流形成早于(或同步于)山地隆起,只要下切速度跟得上抬升,河便能守住原河道、垂直切穿正在升起的构造,形成横切山脉的深峡(而非绕行)。三峡是经典讨论案例。
- 剖面画法: 隆起的褶皱山被一条垂直深切的河道贯穿;两侧山体、谷底河。
- 动画脚本: ①平原上的河→②山体沿河缓慢隆起→③河同步下切、保持原道→④横切成峡。
- 参数槽: [河流, 穿切构造, 抬升与下切关系]
- 易错警示: 先成河 vs 叠置河等成因有别;三峡成因学界仍有讨论,表述留待核对。
- 状态: 起草

## 外力/冰川/U形谷
- 一句话: 冰川以整个横截面刨蚀,把V形河谷拓成U形槽谷。
- 原理讲解: 冰川充满谷地,以底部与两壁同时磨蚀、拔蚀,把河流切出的V形谷改造成宽底陡壁的U形;常过度下切,谷底可低于海平面,支谷被截成高挂的悬谷(悬谷瀑布)。
- 剖面画法: 宽底陡壁的U形横剖面;旁标V形对比;示悬谷。
- 动画脚本: ①V形河谷→②冰川充填→③整截面刨蚀拓宽→④冰退留U形谷+悬谷。
- 参数槽: [谷宽深, 是否过度下切, 悬谷]
- 易错警示: U形是冰刨的,V形是水切的——横剖面形态即指纹。
- 状态: 起草

## 外力/冰川/峡湾
- 一句话: 冰期U形谷深切入海平面以下,冰退海进,海水灌入成峡湾。
- 原理讲解: 沿海冰川把谷底过度下切到海平面以下;冰期结束后冰退、全新世海面上升,海水灌入这些深邃U形谷,成为深而窄、两壁陡立的峡湾,常有冰碛"门槛"。
- 剖面画法: 纵剖面示深谷被海水淹没;口部画冰碛槛;两壁陡峭。
- 动画脚本: ①冰川过度下切→②冰退→③海面上升→④海水灌入成峡湾。
- 参数槽: [深度, 长度, 是否有冰碛槛]
- 易错警示: 峡湾≠普通海湾,核心是"冰川U形谷+海侵"的组合。
- 状态: 起草

## 气候/副高与荒漠
- 一句话: 副热带下沉气流晴朗少雨,造就南北纬约30°一线的荒漠带。
- 原理讲解: 三圈环流中,赤道上升的空气在副热带下沉、增温、相对湿度骤降,长年晴朗少雨;全球大型沙漠(撒哈拉、阿拉伯、澳洲内陆)多沿此带分布。叠加内陆与寒流可更干。
- 剖面画法: 经向环流示意:赤道上升、副热带下沉支落在荒漠带。
- 动画脚本: ①赤道受热上升致雨→②高空向极→③副热带下沉增温→④地表干旱成漠。
- 参数槽: [纬度带, 叠加因素(内陆/寒流)]
- 易错警示: 副热带荒漠是"下沉气流"造成,与极地荒漠(冷干)成因不同。
- 状态: 起草

## 气候/季风
- 一句话: 海陆热力差 + 风带季移,风向随季节反转,旱雨季分明。
- 原理讲解: 夏季陆地增热快、形成低压,风从海洋吹向陆地、带来丰沛降水;冬季相反,风由陆地吹向海洋、干燥少雨。南亚、东亚季风显著,塑造了夏雨集中、冬季干燥(且常带尘)的节律。
- 剖面画法: 夏/冬两幅小图,海陆气压与风向箭头相反。
- 动画脚本: ①夏:陆低压、海风带雨→②冬:陆高压、陆风干燥(切换)。
- 参数槽: [季风区, 夏雨占比, 冬季特征]
- 易错警示: 季风是风向"季节反转",不只是"夏天下雨"这么简单。
- 状态: 起草

## 气候/赤道低压与雨林
- 一句话: 终年上升气流、对流雨日日报到,养出热带雨林。
- 原理讲解: 赤道辐合带终年受热、空气强烈上升,午后对流雨频繁,高温高湿稳定,孕育常绿阔叶雨林与极高生物多样性。水循环强烈,但养分多锁在生物量中,土壤反而贫瘠(见土壤模块)。
- 剖面画法: 赤道上升气流 + 积雨云 + 雨林冠层分层示意。
- 动画脚本: ①强烈受热→②空气上升成云→③午后对流雨→④雨林繁茂。
- 参数槽: [区域, 年降水, 干季有无]
- 易错警示: 雨林肥沃是误解——肥力在树体,土层贫瘠。
- 状态: 起草

## 外力/风/沙丘
- 一句话: 风搬运沙粒遇阻堆积,新月形沙丘随风缓慢迁移。
- 原理讲解: 风以跃移搬运沙粒,遇障碍或风速降低即堆积;迎风坡缓、背风坡陡(休止角),整座沙丘顺主风向缓慢前移。风向单一成新月形,多向成星状等。
- 剖面画法: 缓迎风坡 + 陡背风坡的不对称剖面;箭头示迁移。
- 动画脚本: ①风扬沙→②遇阻堆积→③迎缓背陡成形→④整体顺风迁移。
- 参数槽: [沙丘类型, 主风向, 迁移速率]
- 易错警示: 沙丘不对称剖面即风向指针,迎风缓、背风陡。
- 状态: 起草

## 外力/风/风蚀地貌
- 一句话: 风挟沙磨蚀,雕出雅丹、风蚀蘑菇与风蚀洼地。
- 原理讲解: 近地面挟沙气流像砂纸,对岩土进行磨蚀与吹蚀:软硬差异处磨出沟垄相间的雅丹,基部被磨细成风蚀蘑菇,松散物被吹走留下洼地与砾漠。
- 剖面画法: 顺风向的沟垄(雅丹);蘑菇石下细上粗。
- 动画脚本: ①挟沙风→②差异磨蚀→③雅丹/蘑菇石/洼地成形。
- 参数槽: [地貌类型, 岩性, 主风向]
- 易错警示: 磨蚀集中在近地面(挟沙层),故蘑菇石"细腰"在下部。
- 状态: 起草

## 风化/温差剥离
- 一句话: 昼夜剧烈胀缩使岩石表层成片剥落。
- 原理讲解: 荒漠昼夜温差极大,岩石表里、不同矿物胀缩不一,反复应力使表层呈片状/壳状剥离;干燥少植被环境下尤为显著(常与盐风化协同)。
- 剖面画法: 岩块表层呈层状剥壳;示意昼热夜冷。
- 动画脚本: ①昼受热膨胀→②夜骤冷收缩→③反复→④表层剥落。
- 参数槽: [岩性, 温差幅度]
- 易错警示: 纯温差作用强弱学界有争议,常与水/盐协同,表述待核对。
- 状态: 起草

## 外力/风/黄土堆积
- 一句话: 强风把粉尘搬运千里、岁岁沉降,百万年积成深厚黄土。
- 原理讲解: 冬季风把内陆干旱区的细粉尘长距离搬运,在下风地带持续沉降、堆积成深厚黄土;堆积期与较暖湿的成壤期交替,黄土中夹多层古土壤,成为记录第四纪干湿气候的"天书"。
- 剖面画法: 厚层黄土 + 其中夹多条深色古土壤条带。
- 动画脚本: ①冬季风扬尘→②长距离搬运→③下风沉降堆积→④干湿交替成黄土—古土壤序列。
- 参数槽: [厚度, 物源, 古土壤层数]
- 易错警示: 黄土是"风积",不是流水沉积;古土壤是暖湿期的标志层。
- 状态: 起草

## 外力/流水/曲流与牛轭湖
- 一句话: 平缓河段凹岸蚀、凸岸积,河曲摆动,截弯取直留下牛轭湖。
- 原理讲解: 落差耗尽后河流转为侧蚀:凹岸冲刷成陡坎、凸岸堆积成边滩,弯曲不断加大并向下游迁移;洪水截弯取直后,废弃的旧河弯成为牛轭湖。
- 剖面画法: 平面图为佳:蜿蜒河道 + 凹蚀凸积 + 牛轭湖。
- 动画脚本: ①轻微弯曲→②凹蚀凸积放大弯度→③颈部变窄→④截弯取直、留牛轭湖。
- 参数槽: [河段坡降, 弯曲度, 牛轭湖]
- 易错警示: 曲流多发于坡缓、落差小的河段,不是"水大就弯"。
- 状态: 起草

## 外力/流水/三角洲
- 一句话: 河流入海(湖)流速归零,泥沙分汊堆积成洲。
- 原理讲解: 河水入静水体流速骤降,搬运能力丧失,泥沙在河口分汊、逐层向外淤积成扇形或鸟足状三角洲;沉积由粗到细向海排列,是肥沃平原与重要生境。
- 剖面画法: 河口分汊 + 向海前积的层理(顶积/前积/底积)。
- 动画脚本: ①携沙入海→②流速降、沉积→③分汊延伸→④三角洲外扩。
- 参数槽: [河流, 形态(扇/鸟足), 潮浪影响]
- 易错警示: 三角洲形态受河、浪、潮三者博弈,不只由河决定。
- 状态: 起草

## 土壤/代表土类
- 一句话: 不同气候—生物条件配出不同土:黑土、红壤、荒漠土、冻土各有成因。
- 原理讲解: 成土受气候与生物主导:温带草原腐殖质富集成黑土(肥沃);湿热区强淋溶、铁铝富集成红黄壤(贫瘠,养分在生物量);干旱区有机质少、浅薄多盐成荒漠土;低温区分解慢、有机质封存成(冻)土。
- 剖面画法: 并列若干土壤剖面(O-A-B-C),按气候带标注差异。
- 动画脚本: ①母质起步→②气候/生物作用(淋溶/积累)→③分异出不同土类。
- 参数槽: [土类, 气候带, 肥力特征]
- 易错警示: 湿热≠肥沃:强淋溶反而使红黄壤贫瘠。
- 状态: 起草

## 外力/海岸/珊瑚礁
- 一句话: 珊瑚虫骨骼世代堆积造礁,随海面/岛屿变化由岸礁→堡礁→环礁。
- 原理讲解: 造礁珊瑚需温暖、清澈、有光的浅海;其碳酸钙骨骼世代堆积成礁(本质是生物灰岩)。岛屿沉降或海面上升时,礁体向上追长,经岸礁、堡礁、环礁演化(达尔文沉降说)。海面下降则礁出露死亡。
- 剖面画法: 岸礁/堡礁(礁—潟湖—岸)/环礁三型对比;礁向上追长箭头。
- 动画脚本: ①浅海珊瑚造礁→②基底沉降/海面升→③礁向上追长→④岸礁→堡礁→环礁。
- 参数槽: [礁型, 所处海面阶段, 基底升降]
- 易错警示: 珊瑚礁是生物造的灰岩;海面下降会让它出露死亡,不是"长得更高"。
- 状态: 起草
