## 第1类 · 岩石循环(补全 9)

### 岩石/循环总图
- 一句话: 岩浆—火成—沉积—变质相互转化的总框架,全库的"地图"。
- 原理讲解: 岩浆冷凝成火成岩;岩石暴露地表被风化剥蚀、搬运沉积、固结成沉积岩;埋深受高温高压成变质岩;再深处重熔回岩浆——任一岩类都能转成另一类,循环往复。
- 剖面画法: 环形箭头图,四节点(岩浆/火成/沉积/变质)+ 转化条件标注。
- 动画脚本: 沿环走一圈,逐段点亮转化条件。
- 参数槽: [起点岩类]
- 易错警示: 不是单向链,是可逆循环(沉积岩也可直接变质或重熔)。
- 状态: 起草

### 岩石/火成/流纹岩
- 一句话: 酸性岩浆黏稠含气多,喷发剧烈,冷却成浅色细粒岩。
- 原理讲解: 富硅、黏度高,气体难逸→爆发式喷发(火山灰、浮岩、火山碎屑流),熔岩流动差;与花岗岩同成分,是其喷出版。
- 剖面画法: 浅色细粒,常伴火山碎屑层。
- 动画脚本: ①黏稠岩浆憋压→②爆发→③灰与浮岩堆积。
- 参数槽: [喷发风格, 时代]
- 易错警示: 流纹岩=花岗岩的喷出对应物(成分同、产状异)。
- 状态: 起草

### 岩石/火成/花岗岩
- 一句话: 酸性岩浆地下缓慢冷却,粗粒浅色,大陆地壳与造山带的核心岩石。
- 原理讲解: 富硅铝岩浆侵入地壳深处,慢冷使石英、长石、云母长成肉眼可见的粗晶;常成大型岩基,是大陆地壳代表岩石,出露需上覆剥蚀。
- 剖面画法: 粗晶斑杂纹,作岩基置于造山带核部。
- 动画脚本: ①岩浆侵位→②慢冷结晶→③上覆剥蚀、岩基出露。
- 参数槽: [岩体规模, 时代]
- 易错警示: 花岗岩是侵入(深成)岩、粗粒;与流纹岩成分同而冷却环境异。
- 状态: 起草

### 岩石/火成/辉长岩
- 一句话: 基性岩浆在地下缓冷,与玄武岩同成分、粗粒,是洋壳下部主要岩石。
- 原理讲解: 化学成分与玄武岩相同,但侵入慢冷→粗粒暗色;构成洋壳下层。
- 剖面画法: 暗色粗粒,置于洋壳下层。
- 动画脚本: ①基性岩浆侵入→②慢冷成粗晶。
- 参数槽: [产状, 时代]
- 易错警示: 辉长岩(深成)≠玄武岩(喷出),成分同、冷却异。
- 状态: 起草

### 岩石/火成/侵入产状
- 一句话: 岩基、岩株、岩墙、岩床——岩浆在地下占位的几种形态。
- 原理讲解: 顺层贯入为岩床、切层贯入为岩墙、巨型不规则深成体为岩基、较小为岩株;产状决定后期出露与地貌。
- 剖面画法: 剖面示各类产状与围岩关系。
- 动画脚本: 岩浆沿不同路径侵位、占据各产状。
- 参数槽: [产状类型]
- 易错警示: 岩床顺层、岩墙切层,勿混。
- 状态: 起草

### 岩石/沉积/蒸发岩
- 一句话: 封闭水体蒸发浓缩,析出岩盐、石膏,指示干旱古环境。
- 原理讲解: 干旱区封闭/半封闭水体强烈蒸发,盐类按溶解度先后析出(石膏→岩盐→钾盐)成层;是古干旱气候的直接标志与盐矿来源。
- 剖面画法: 浅色层,标盐类析出序列。
- 动画脚本: ①水体蒸发→②浓缩→③盐类逐级析出成层。
- 参数槽: [盐类组合, 时代]
- 易错警示: 蒸发岩=干旱封闭环境标志,与煤(湿热)相反。
- 状态: 起草

### 岩石/沉积/煤
- 一句话: 植物遗体在沼泽中堆积、埋藏、碳化,指示古代湿热森林。
- 原理讲解: 湿热森林沼泽缺氧水下植物遗体堆成泥炭,埋深增温增压逐步碳化为褐煤→烟煤→无烟煤;记录成煤期的湿热富植被环境。
- 剖面画法: 黑色煤层夹于沉积序列。
- 动画脚本: ①植物堆积→②泥炭→③埋藏碳化→④煤。
- 参数槽: [煤级, 成煤期]
- 易错警示: 煤指示湿热富植被,与蒸发岩(干旱)环境相反。
- 状态: 起草

### 岩石/沉积/层理与化石
- 一句话: 沉积岩的"页码与文字":层理记录环境,化石记录年代与生境。
- 原理讲解: 层理(水平、交错、粒序)记录沉积时的水/风动力与环境;化石既能定相对年代(标准化石)又能复原古生境;二者合读即把岩层翻译成环境史。
- 剖面画法: 示不同层理型 + 化石符号。
- 动画脚本: ①沉积留下层理→②生物埋藏成化石→③后人据此读环境。
- 参数槽: [层理型, 化石组合]
- 易错警示: 交错层理可判古水流/风向;粒序层理指示水动力变化。
- 状态: 起草

### 岩石/变质/接触变质
- 一句话: 岩浆"烤"围岩:灰岩→大理岩,砂岩→石英岩。
- 原理讲解: 侵入体周围围岩受高温(压力次要)局部改造,形成环状变质晕;灰岩重结晶成大理岩,砂岩成石英岩,范围有限。
- 剖面画法: 岩体周围一圈变质晕。
- 动画脚本: ①岩浆侵入→②烘烤围岩→③接触带重结晶。
- 参数槽: [原岩, 变质矿物]
- 易错警示: 接触变质靠"热"、范围小;区域变质靠"高温高压"、范围大。
- 状态: 起草

## 第2类 · 构造变形(补全 2)

### 构造/走滑断层
- 一句话: 两盘水平错动,地表错开河道与山脊。
- 原理讲解: 剪切应力下两盘沿近直立断面水平相对滑动(左旋/右旋),常构成转换边界(圣安德烈斯);地表水系、田埂被同步错断,是判别依据。
- 剖面画法: 平面图为佳,河道被水平错开。
- 动画脚本: ①剪切积累→②两盘水平错动→③地物错移。
- 参数槽: [旋向, 错距]
- 易错警示: 走滑以水平位移为主、垂直起伏小;判别看地物平面错断。
- 状态: 起草

### 构造/节理
- 一句话: 岩石中无明显位移的裂隙,风化与水流沿它"下刀"。
- 原理讲解: 岩石受力或冷却收缩产生的裂隙,两侧无明显相对位移(区别于断层);成组节理控制风化、溶蚀、侵蚀的优先通道(喀斯特、柱状节理、岩崩皆沿它)。
- 剖面画法: 岩体中成组平行/网状细缝。
- 动画脚本: ①受力/收缩→②裂隙张开→③外力沿缝加工。
- 参数槽: [成因, 密度, 方向]
- 易错警示: 节理无位移(≠断层),但它是外力作用的"导引线"。
- 状态: 起草

## 第3类 · 板块运动(补全 5)

### 板块/总论
- 一句话: 岩石圈分块缓慢运动,边界类型决定地震、火山与大地貌格局。
- 原理讲解: 地球外层岩石圈裂成若干板块,驮在软流圈上由地幔对流等驱动而移动;板块边界(离散/汇聚/转换)集中绝大多数地震火山,板内相对稳定(热点例外)。
- 剖面画法: 全球板块与三类边界示意。
- 动画脚本: ①地幔对流→②板块移动→③边界活动(地震火山)。
- 参数槽: [—]
- 易错警示: 是"岩石圈板块"在动,不是地壳单独动;板内也有热点活动。
- 状态: 起草

### 板块/汇聚/洋洋俯冲
- 一句话: 大洋板块俯冲到另一洋块之下,生成海沟 + 岛弧。
- 原理讲解: 两大洋板块相遇,较老较冷较重者下插,形成深海沟与后方火山岛弧(马里亚纳等);机制同洋陆俯冲,差别在上覆为洋壳。
- 剖面画法: 海沟 + 下插板片 + 岛弧火山。
- 动画脚本: ①一洋板块下插→②脱水熔融→③岛弧火山。
- 参数槽: [俯冲对, 岛弧]
- 易错警示: 与洋陆俯冲同理,差别只在上覆板块性质(洋vs陆)。
- 状态: 起草

### 板块/转换边界
- 一句话: 板块水平剪切滑过,多地震少火山。
- 原理讲解: 两板块既不生长也不消亡,沿转换断层水平错动;应变积累突然释放成强震,少有岩浆活动。
- 剖面画法: 两板块沿断层反向滑动箭头。
- 动画脚本: ①剪切积累→②突破错动→③地震。
- 参数槽: [代表断层]
- 易错警示: 转换边界以地震为主、火山少,区别于另两类边界。
- 状态: 起草

### 板块/火山/破火山口
- 一句话: 喷空岩浆房后山体塌陷,留下巨型洼地,常积水成湖。
- 原理讲解: 大规模喷发抽空浅部岩浆房,失去支撑的山顶/山体塌陷,形成远大于普通火山口的环形洼地(caldera);积水即火口湖(如长白山天池)。
- 剖面画法: 塌陷环形洼地,内积湖。
- 动画脚本: ①巨喷→②岩浆房空→③顶部塌陷→④积水成湖。
- 参数槽: [直径, 是否积水]
- 易错警示: 破火山口由"塌陷"形成,不是被炸出的大坑。
- 状态: 起草

### 板块/地震
- 一句话: 断层突然错动释放应变能;震源、震级与破坏的基本逻辑。
- 原理讲解: 构造应力使断层两盘积累弹性应变,超过摩擦极限即突然错动、释放能量成地震波;震源深度、震级与烈度是基本量,俯冲带可有深源地震。
- 剖面画法: 断层错动 + 波纹扩散 + 震源/震中标注。
- 动画脚本: ①应变积累→②突破→③错动放波。
- 参数槽: [震级, 深度, 断层型]
- 易错警示: 震级(能量)与烈度(地表破坏)是两回事。
- 状态: 起草

## 第4类 · 风化(补全 5)

### 风化/冻融
- 一句话: 水渗入裂隙结冰膨胀,反复冻融把岩石劈碎。
- 原理讲解: 水结冰体积增约9%,在裂隙中像楔子反复撑裂岩石;高山高纬冻融频繁带最盛,产出大片碎石坡(倒石堆)。
- 剖面画法: 岩裂隙中冰楔 + 坡脚碎石堆。
- 动画脚本: ①水入缝→②结冰膨胀→③反复→④岩石崩裂。
- 参数槽: [冻融频次]
- 易错警示: 物理风化(机械破碎),不改变矿物成分。
- 状态: 起草

### 风化/盐风化
- 一句话: 盐分在孔隙中结晶撑裂岩石,干旱区与海岸常见。
- 原理讲解: 含盐水进入岩石孔隙,蒸发后盐结晶膨胀产生应力,反复使岩石崩解、成蜂窝状;以物理为主、化学协同。
- 剖面画法: 岩面蜂窝状孔洞。
- 动画脚本: ①含盐水入孔→②蒸发结晶→③撑裂剥落。
- 参数槽: [盐源, 环境]
- 易错警示: 盐风化是结晶"撑裂"(物理),不是盐"溶解"岩石。
- 状态: 起草

### 风化/氧化
- 一句话: 含铁矿物"生锈",岩土变红。
- 原理讲解: 矿物中的铁等在有氧有水环境下氧化成氧化铁,使岩土染红黄并变疏松;红壤、丹霞红层的色调多源于此。
- 剖面画法: 岩土染红/黄。
- 动画脚本: ①含铁矿物遇氧水→②氧化→③变红疏松。
- 参数槽: [原矿物, 环境]
- 易错警示: 红色来自氧化铁,湿热氧化强→红壤;非"被烤红"。
- 状态: 起草

### 风化/水解
- 一句话: 长石等矿物遇水分解成黏土,岩石由硬变酥。
- 原理讲解: 硅酸盐(尤其长石)与含弱酸的水反应,分解为黏土矿物并析出可溶离子;是化学风化主力,把硬岩变成松软风化壳与土壤母质。
- 剖面画法: 岩石表层化为黏土/风化壳。
- 动画脚本: ①长石+水→②分解为黏土+溶液→③风化壳加厚。
- 参数槽: [原矿物, 气候]
- 易错警示: 水解是化学分解成新矿物(黏土),不是单纯泡软。
- 状态: 起草

### 风化/生物风化
- 一句话: 根劈、穴居与生物酸,生命参与拆解岩石。
- 原理讲解: 植物根撑裂岩缝(物理),根与微生物分泌酸促进分解(化学),动物穴居翻动——生物以物理化学两途加速风化与成土。
- 剖面画法: 根系入岩缝 + 地衣覆岩。
- 动画脚本: ①根扎入→②撑裂/分泌酸→③岩石松解。
- 参数槽: [生物类型]
- 易错警示: 生物风化兼有物理与化学两类作用。
- 状态: 起草

## 第5类 · 外营力地貌(补全 8)

### 外力/流水/瀑布与裂点
- 一句话: 软硬岩差异侵蚀成陡坎,瀑布不断后退。
- 原理讲解: 河床纵剖面上软硬岩交界或构造处形成陡坎(裂点),硬岩护顶、软岩被掏空,瀑布沿陡坎向上游溯源后退(尼亚加拉型)。
- 剖面画法: 河床纵剖面陡坎 + 后退箭头。
- 动画脚本: ①软岩掏空→②硬岩崩落→③裂点上溯后退。
- 参数槽: [岩性差异, 后退速率]
- 易错警示: 瀑布会向上游后退(溯源侵蚀),不是固定不动。
- 状态: 起草

### 外力/流水/冲积扇
- 一句话: 山口处水流骤缓,砾砂呈扇形堆积。
- 原理讲解: 河流出山口坡度骤减、水流分散、搬运力骤降,粗碎屑在山前堆成扇形;干旱区冲积扇是绿洲与含水层所在。
- 剖面画法: 平面扇形 + 剖面由粗到细向外。
- 动画脚本: ①出山口→②流缓分汊→③扇形堆积。
- 参数槽: [规模, 物源]
- 易错警示: 冲积扇在山口(陆上),三角洲在入水口,别混。
- 状态: 起草

### 外力/流水/河流阶地
- 一句话: 抬升与下切交替,把旧河漫滩抬成沿河的台阶。
- 原理讲解: 区域抬升或气候变化使河流重新下切,原河漫滩被抬高、悬于新河床之上成阶地;多级阶地记录多次旋回,越高越老。
- 剖面画法: 河谷两侧多级台阶,标级序。
- 动画脚本: ①河漫滩→②下切→③旧滩成阶→④再循环。
- 参数槽: [阶地级数, 对应事件]
- 易错警示: 阶地越高越老;它是"抬升+下切"的记录器。
- 状态: 起草

### 外力/冰川/冰斗角峰刃脊
- 一句话: 雪线附近冰川刨蚀成围椅状洼地,夹峙出角峰与刃脊。
- 原理讲解: 山地冰川源头反复刨蚀成围椅状冰斗;相邻冰斗后壁后退相夹成锐利刃脊,三个以上汇聚削出尖锐角峰(马特洪峰型)。
- 剖面画法: 冰斗 + 刃脊 + 角峰组合。
- 动画脚本: ①冰斗刨蚀后退→②刃脊变锐→③角峰成形。
- 参数槽: [地貌组合]
- 易错警示: 这些是冰川"刨蚀"地貌(高山区),非堆积地貌。
- 状态: 起草

### 外力/冰川/冰碛地貌
- 一句话: 冰川搬来的混杂碎屑就地卸货:终碛垄、侧碛、漂砾。
- 原理讲解: 冰川像传送带搬运无分选碎屑(冰碛),消融处卸下:末端堆终碛垄、两侧成侧碛、底部铺底碛,巨大外来石块成漂砾;终碛常拦水成湖。
- 剖面画法: 冰川末端终碛垄 + 侧碛 + 漂砾。
- 动画脚本: ①冰川搬运→②消融卸货→③各类碛垄成形。
- 参数槽: [碛型, 是否成湖]
- 易错警示: 冰碛无分选(大小混杂),区别于流水/风的分选沉积。
- 状态: 起草

### 外力/海岸/海蚀地貌
- 一句话: 浪打基岩岸,海蚀崖后退,留下海蚀拱与海蚀柱。
- 原理讲解: 波浪与挟带砾石持续冲蚀基岩岸,在浪基面附近掏出海蚀穴、崖体崩退;岬角差异侵蚀留海蚀拱、海蚀柱与浪蚀台。
- 剖面画法: 海蚀崖 + 浪蚀台 + 海蚀柱。
- 动画脚本: ①浪掏蚀穴→②崖崩后退→③拱→柱。
- 参数槽: [岩性, 地貌型]
- 易错警示: 海蚀以波浪机械冲蚀为主,多见于基岩陡岸。
- 状态: 起草

### 外力/海岸/海积地貌
- 一句话: 浪与沿岸流搬沙堆积:沙滩、沙嘴、潟湖。
- 原理讲解: 波浪与沿岸流把泥沙沿岸搬运、在能量低处堆积成沙滩,在岬湾/河口延伸成沙嘴、连岛坝,封闭海湾成潟湖。
- 剖面画法: 沙滩/沙嘴/潟湖平面示意。
- 动画脚本: ①沿岸流输沙→②低能处堆积→③沙嘴延伸封湾。
- 参数槽: [地貌型, 沙源]
- 易错警示: 海积靠沿岸搬运堆积,与海蚀(冲蚀)相对。
- 状态: 起草

### 外力/块体/滑坡崩塌
- 一句话: 重力直接搬运:斜坡整体下滑或岩体坠落,常由水与地震触发。
- 原理讲解: 重力作用下,饱水、地震或坡脚掏蚀触发斜坡物质整体下滑(滑坡,有滑动面)或陡崖岩块骤然坠落(崩塌)。
- 剖面画法: 滑坡(弧形滑面+滑体)/崩塌(陡崖落石)。
- 动画脚本: ①触发→②失稳→③整体下滑或坠落。
- 参数槽: [类型, 触发因素]
- 易错警示: 滑坡有滑动面、整体移动;崩塌是骤然坠落,二者有别。
- 状态: 起草

## 第6类 · 气候与大气(补全 7)

### 气候/三圈环流
- 一句话: 赤道到极地的气压带与风带,全球气候分布的总骨架。
- 原理讲解: 辐射纬度差驱动低纬、中纬、极地三圈环流,形成赤道低压、副热带高压、副极地低压、极地高压相间的气压带与信风、西风、极地东风风带;气压带风带随季节移动。
- 剖面画法: 半球经向三圈 + 气压带风带标注。
- 动画脚本: ①受热差→②三圈环流→③气压带风带成形。
- 参数槽: [—]
- 易错警示: 气压带风带随太阳直射点季节移动,非固定。
- 状态: 起草

### 气候/柯本分类
- 一句话: 按温度降水把全球气候划成可比类型,"点哪儿都有"的底图。
- 原理讲解: 以月均温与降水阈值把气候分为热带(A)、干旱(B)、温带(C)、冷温带(D)、极地(E)及亚类;是全球可获取、可点击的气候底图。
- 剖面画法: 分类色块 + 判别阈值表。
- 动画脚本: ①取温降数据→②按阈值判别→③落入类型。
- 参数槽: [当地类型]
- 易错警示: 柯本按"温度+降水"统计划界,非按纬度硬分。
- 状态: 起草

### 气候/西风与温带气候
- 一句话: 中纬西风输送水汽与锋面,温带气候的发动机。
- 原理讲解: 中纬盛行西风挟带海洋水汽并伴气旋锋面活动,带来频繁降水与多变天气;大陆西岸尤受其惠(温带海洋性)。
- 剖面画法: 中纬西风 + 锋面气旋示意。
- 动画脚本: ①西风输汽→②锋面过境→③降水。
- 参数槽: [区域, 海陆位置]
- 易错警示: 西风带天气多变,降水主要靠锋面气旋而非地形。
- 状态: 起草

### 气候/地形雨影
- 一句话: 迎风坡抬升致雨,背风坡下沉干燥,一山隔出两种气候。
- 原理讲解: 气流被山地抬升冷却凝结,迎风坡多雨;翻脊后下沉增温、湿度骤降,背风坡成干燥雨影区(喜马拉雅南湿、藏南北干)。
- 剖面画法: 山体两侧:迎风云雨、背风晴干。
- 动画脚本: ①气流上坡致雨→②翻山下沉→③背风干燥。
- 参数槽: [山体, 迎背风差]
- 易错警示: 雨影区干燥源于"下沉增温",不只是水汽被山挡。
- 状态: 起草

### 气候/海拔分带
- 一句话: 每升约千米降温约6.5℃,高山把多个气候带垂直叠放。
- 原理讲解: 气温随海拔递减(约6.5℃/千米),使高大山体从山麓到山顶出现类似纬度变化的气候植被带谱;迎背风、坡向使带谱不对称。
- 剖面画法: 山体侧面垂直带谱(基带→雪线)。
- 动画脚本: ①沿坡上升→②逐带变冷→③垂直分带。
- 参数槽: [基带, 雪线高度]
- 易错警示: 垂直带谱受纬度、坡向、干湿共同影响,不只看高度。
- 状态: 起草

### 气候/洋流
- 一句话: 暖流增温增湿、寒流降温减湿,改写沿岸气候剧本。
- 原理讲解: 表层洋流由风与地转驱动成环流;暖流沿岸增温增湿(西欧),寒流沿岸降温减湿、甚至助成沿岸荒漠(秘鲁、纳米布)。
- 剖面画法: 洋流箭头(暖红寒蓝)+ 沿岸气候批注。
- 动画脚本: ①洋流抵岸→②改变沿岸温湿。
- 参数槽: [暖/寒流, 沿岸效应]
- 易错警示: 寒流沿岸常更干(助成荒漠),暖流沿岸更湿。
- 状态: 起草

### 气候/大陆性与海洋性
- 一句话: 离海远近决定温差与降水节律,内陆"冬冷夏热"的根源。
- 原理讲解: 海洋热容大、调节强,沿海温差小、降水匀(海洋性);深居内陆调节弱,冬冷夏热、温差大、降水少而集中(大陆性)。
- 剖面画法: 沿海 vs 内陆年温曲线对比。
- 动画脚本: ①同纬度由海向陆→②温差逐渐增大。
- 参数槽: [距海, 温差]
- 易错警示: 大陆性核心是"温差大、节律剧",常伴较干但不等于干旱。
- 状态: 起草

## 第7类 · 土壤(补全 2)

### 土壤/成土五因素
- 一句话: 母质、气候、生物、地形、时间共同"配方"出一方土。
- 原理讲解: 母质提供矿物原料,气候(温湿)主导风化淋溶强度,生物贡献有机质与养分循环,地形影响水热与侵蚀,时间决定发育程度——五者耦合决定土壤类型与肥力。
- 剖面画法: 五因素箭头共同指向一个土壤剖面。
- 动画脚本: ①母质起步→②五因素作用→③成土。
- 参数槽: [主导因素]
- 易错警示: 气候与生物常主导,但五因素缺一不可。
- 状态: 起草

### 土壤/剖面分层
- 一句话: O-A-B-C 层:从枯枝落叶到风化母质的垂直序列。
- 原理讲解: 自上而下 O(枯落物)、A(腐殖质淋溶层)、B(淀积层)、C(风化母质)、R(基岩);层位发育程度反映成土强度与时间。
- 剖面画法: 竖向土壤剖面分层标注。
- 动画脚本: ①母质→②自上而下逐层发育→③成熟剖面。
- 参数槽: [层位厚度, 发育度]
- 易错警示: A层淋溶、B层淀积,勿颠倒。
- 状态: 起草

## 第8类 · 古地理(补全 2)

### 古地理/超大陆旋回
- 一句话: 大陆聚合—裂解—再聚合的亿年级循环,大地理格局之源。
- 原理讲解: 大陆在数亿年尺度聚成超大陆(如盘古)、再裂解漂移、再重新聚合(威尔逊旋回);决定海陆轮廓、洋流、气候与生物分布的长周期变迁。
- 剖面画法: 几帧古大陆位置(聚合→裂解→再聚)。
- 动画脚本: 大陆漂移、聚散循环。
- 参数槽: [关键时代, 超大陆名]
- 易错警示: 这是亿年尺度循环,与板块当下运动是同一机制的长时段表现。
- 状态: 起草

### 古地理/古环境证据
- 一句话: 化石、沉积相、岩性如何告诉我们"这里以前是什么"。
- 原理讲解: 海相化石与灰岩指示曾为海,煤指示湿热森林沼泽,蒸发岩指示干旱,冰碛岩指示冰期,交错层理指示风/水流向——把证据拼起即可复原古环境与变迁。
- 剖面画法: 证据清单 → 古环境推断的对照表。
- 动画脚本: ①收集各类证据→②拼出古场景。
- 参数槽: [证据类型]
- 易错警示: 古地理是"由证据反推",每条结论都要有岩石/化石支撑。
- 状态: 起草
